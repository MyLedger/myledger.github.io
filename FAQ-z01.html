版本介紹 / 新功能

1.5.0 ( 2018-05-02 )

  • 電子發票掃描
  • 手機條碼管理
  • 看廣告兌換付費功能
  • UI/UX 優化

1.4.10 ( 2018-03-05 )

  • 樣板定期預約交易新增：每兩個月 / 每季 / 每半年
  • 依預算查看收支紀錄

  • 資料庫備份/還原
  • 樣板管理
  • 設定 - 每日記帳提醒、每月 / 週起始日、首頁設定、語言設定(新增簡體中文)、常用分類 / 帳戶設定、清空所有紀錄、安全鎖(指紋辨識)
  • 搜尋功能
  • 新增使用指南
  • 累計支出
  • 帳戶 - 顯示即時餘額、設定結帳日
  • 長按拖曳更換排序
  • 轉帳 - 新增手續費
  • UI/UX 優化
